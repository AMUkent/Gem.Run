# Gem.Run
2504040020 DGD104.2 Final Project 
Concept: Gem run is a game where the player is to collect a target amount of gems before the time runs out. The player also needs to avoid obstacles while doing this or they will loose hearts.
Core Loop:Player moves; collects gems; dodges obstacles; catches target amount of gems; timer runs out; player wins; moves on
Win: Player catches target amount of gems before time ends
Lose: Timer runs out and player doesn't reach target/ player hits too many obstacles and runs out of hearts.
Scene:MainMenu,Game,WinScreen,LoseScreen
Controls:W-Forward;D-Right;A-Left;S-Back
